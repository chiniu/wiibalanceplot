wiiblanceplot
=============

What is this
------------
Use wii blance board to track barycentre.
Combine below two 

* GUI [scalesgui.py](http://abstrakraft.org/cwiid/ticket/63)

* BT communication use [wiiboard-simple](https://code.google.com/p/wiiboard-simple/)

Requirements
-----------
* python-pygame
* python-bluez
* bluez-utils
* Bluetooth

Usage
-----
 
    ./blanceplot.py
